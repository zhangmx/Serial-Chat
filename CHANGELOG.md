## [1.0.0] - 2026-01-05

### Added
- Initial release of Serial Chat
- Serial port communication with chat-like interface
- Message display in Text or Hex format
- Friend list with online/offline status indicators
- Group chat functionality with message forwarding
- Message history persistence
- Console dock widget for debug output
- Connect All menu option to open all ports
- Group message target selection
- Application icon (SVG)
- Interactive title and status buttons in chat header
- Case-insensitive search for ports and groups
- Export message history to JSON
- Cross-platform build support (Windows, macOS, Linux)
- GitHub Actions CI/CD workflow
- VSCode workspace configuration

### Technical Features
- Qt 5.12+ support
- CMake build system
- Google Test unit testing framework
- Semantic versioning
- Comprehensive documentation
